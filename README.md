# Sylin Homebrew Tap

Homebrew formulae for [Sylin](https://github.com/sylin-org) tools.

## Ghostlight

Governed browser automation over your own authenticated Chromium session, for AI agents
(any MCP client, your real logged-in browser, capability grants + audit when you want them).

```sh
brew install sylin-org/tap/ghostlight
ghostlight install   # connect the browser side (idempotent)
```

Docs and the full walkthrough: https://sylin-org.github.io/ghostlight/
